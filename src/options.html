Notion API設定

Google連携設定 (任意)

Googleカレンダー(Tasks)に「今日のタスク」として追加するための設定です。
Google Apps Script (GAS) のWebアプリURLを入力してください。

ℹ️ GAS設定手順 (クリックして展開)
  1. Google Apps Script で「新しいプロジェクト」を作成
  2. エディタに以下のコードをコピペ 
    function doPost(e) {
  var data = JSON.parse(e.postData.contents);
  var task = { title: data.title, due: new Date().toISOString() };
  var newTask = Tasks.Tasks.insert(task, '@default');
  return ContentService.createTextOutput(JSON.stringify({status: 'success'}));
}
  3. 左側の「サービス」(+) から Google Tasks API を追加
  4. 「デプロイ」→「新しいデプロイ」→「Webアプリ」を選択
  5. アクセスできるユーザーを 「全員」 にしてデプロイ
  6. 発行されたURLを上のフォームに入力して保存

データベース設定

表示を切り替えたいデータベースを複数登録できます。

📚 セットアップ手順

  1. Notion Integrations で新しいインテグレーションを作成
  2. 内部インテグレーション・トークンを上記のAPIキー欄に入力して保存
  3. TODO用データベースをNotionで開き、右上の「...」から作成したコネクトを追加
  4. URLの「?v=」の前の32文字（データベースID）をコピーして、ここに追加
