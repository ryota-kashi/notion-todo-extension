# プライバシーポリシー

**Notion TODO Manager**（以下、「本拡張機能」）は、利用者のプライバシーを尊重し、個人情報の保護に努めます。本プライバシーポリシーでは、本拡張機能が収集、利用、管理する情報について説明します。

## 1. データの収集と利用

本拡張機能は、以下の情報を利用者のデバイス上でのみ管理し、開発者が閲覧・収集することはありません。

- **Notion API キー（内部インテグレーション・トークン）**: 利用者のNotionデータベースにアクセスするためにのみ使用されます。
- **Notion データベース ID**: 指定されたデータベースからタスク情報を取得するために使用されます。
- **タスクの内容**: 表示、追加、編集、削除のためにNotion APIを介してやり取りされます。

## 2. データの転送

本拡張機能が取得したデータは、Notionの公式API（`https://api.notion.com`）を介して、利用者のNotionアカウントとの間でのみ直接転送されます。本拡張機能が第三者のサーバーにデータを送信したり、蓄積したりすることはありません。

## 3. ストレージの利用

本拡張機能は、設定情報（APIキーやデータベースID）を保存するために、Chromeの `storage.local`（または `storage.sync`）を使用します。このデータは利用者のブラウザ内に保存されます。

## 4. パーミッションの利用目的

- **sidePanel**: ブラウザのサイドパネルにユーザーインターフェースを表示するために使用します。
- **storage**: 設定情報を保存するために使用します。
- **host_permissions (https://api.notion.com/*)**: Notion APIとの通信を行うために使用します。

## 5. ポリシーの変更

本プライバシーポリシーは、必要に応じて更新されることがあります。重要な変更がある場合は、拡張機能の更新情報等を通じて通知します。
