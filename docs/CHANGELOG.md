# Notion TODO Manager - 変更履歴

## v1.0.0 (2026-01-30)

### 🎉 初回リリース

#### コア機能

- ✅ Notionデータベースとの連携
- ✅ サイドパネルでのTODO表示
- ✅ タスクの完了/未完了切り替え
- ✅ 新規タスク追加
- ✅ リフレッシュ機能

#### UI/UX機能

- ✅ モダンなグラデーションデザイン（紫→青）
- ✅ 完了タスクの削除アニメーション
  - チェックマーク表示（600ms）
  - フェード＆スライドアウト（400ms）
  - 自動削除
- ✅ スムーズなホバーエフェクト
- ✅ レスポンシブデザイン

#### プロパティ表示

- ✅ 期限プロパティ表示
  - 「今日」「明日」の特別表示
  - 期限切れタスクの赤色強調
  - 完了タスクは半透明
- ✅ タグプロパティ表示
  - Multi-select / Select 対応
  - 緑色のバッジデザイン
  - 複数タグ対応

#### 設定機能

- ✅ 設定ページ（APIキー・データベースID入力）
- ✅ ヘッダーの⚙️ボタンで簡単アクセス
- ✅ `chrome.storage.sync`による安全な保存
- ✅ 入力バリデーション

#### セキュリティ

- ✅ APIキーはコードにハードコーディングせず設定ページで入力
- ✅ Chromeの暗号化ストレージに保存
- ✅ データベースIDの自動フォーマット（UUID形式）

#### 対応プロパティ名

**必須:**

- タイトル: `Name`, `Title`, `タイトル`
- 完了: `Done`, `完了`, `Status`

**オプション:**

- 期限: `Due`, `期限`, `DueDate`, `Due Date`
- タグ: `Tags`, `タグ`, `Tag`, `Labels`

#### 技術スタック

- Manifest V3
- Side Panel API
- Notion REST API (v2022-06-28)
- Chrome Storage API
