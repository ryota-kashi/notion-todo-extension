# Chrome Web Store 申請用情報（プライバシーへの取り組み）

Chrome Web Storeの管理画面「プライバシーへの取り組み」タブで要求される項目への回答例です。以下の内容をコピー＆ペーストしてご利用ください。

## 1. 権限の使用理由

### sidePanel (サイドパネル)

> NotionのTODOリストをブラウザのサイドバーに常駐させ、他のタブで閲覧・作業をしながらシームレスにタスクの確認、完了、編集を行えるようにするため。

### storage (ストレージ)

> 利用者のNotion APIキー、データベースID、および表示設定（ダークモードのオンオフ、ソート順、フィルタ設定など）をブラウザ内に安全に保存し、拡張機能の利便性を維持するため。

### ホスト権限 (https://api.notion.com/*)

> Notionの公式APIと通信を行い、ユーザーが指定したデータベースからタスク情報を取得、追加、更新、削除する機能を実行するため。

## 2. リモートコードの使用について

> 本拡張機能はリモートコードを使用していません。すべての実行ロジックは拡張機能パッケージ内に含まれるローカルなスクリプトによって構成されています。

## 3. 単一用途の説明 (Single Purpose)

> NotionのTODOデータベースをChromeのサイドパネルで表示・管理することに特化した、シンプルで軽量なタスク管理ツールです。

## 4. ユーザーデータの取り扱いに関する表明

Googleのデベロッパー プログラム ポリシーに準拠していることを表明するチェックボックス等がある場合、以下の内容を念頭にチェックを入れてください：

- **販売しない**: ユーザーのデータを第三者に販売しません。
- **用途外で使用しない**: 拡張機能の主要な機能（Notion管理）に関係のない目的でデータを使用（または転送）しません。
- **信用情報の収集に使用しない**: ユーザーの信用調査や融資目的でデータを使用しません。

---

**補足**:

- ステータスバーやダッシュボードで「データの収集」について聞かれた場合は、「なし（No）」または「設定情報のみ（APIキー等）」と回答してください。
- ユーザーデータは利用者のNotionアカウントとブラウザ間でのみやり取りされ、開発者には一切送信されません。
